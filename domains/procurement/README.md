# Procurement Coordination

## Coordination Gap
Public procurement opportunities not aggregated.
Bid evaluation processes opaque. Vendor registration duplicated.
SME access to government contracts limited by information asymmetry.

## Data Sources
- Kenya Public Procurement Regulatory Authority (PPRA) — public data
- Open Contracting Data Standard (OCDS) format adoptions
- World Bank procurement data (open)

## MCP Integration
- **fomu-mcp** (active) — forms and procurement

## Key Metrics
- Procurement cycle time (tender → award → delivery)
- SME contract win rate (% of contracts won by SMEs)
- Vendor registration time
- Tender publication to close window (minimum competitive participation time)
- Payment lag (delivery → payment)

## Coordination Workflows
1. Tender aggregation: PPRA scrape → structured alerts → vendor notification
2. Bid preparation: fomu-mcp → bid document assembly assistance
3. Vendor registry: unified registration → multi-entity submission
