# Education Coordination

## Coordination Gap
School enrollment siloed by county. Teacher deployment mismatched to enrollment.
Learning outcomes not tracked longitudinally. Resource distribution not optimized.

## Data Sources
- Kenya National Examinations Council (KCPE/KCSE results, public)
- UNESCO Education data (open)
- UNICEF MICS (open) — education indicators
- World Bank Education Statistics (open)

## MCP Integration
- **elimu-mcp** (active on PyPI) — education coordination

## Key Metrics
- Teacher-to-student ratio by school and county
- Transition rate (primary → secondary)
- Learning outcome index (normalized exam performance)
- School resource index (books, toilets, water)
- Distance to nearest secondary school

## Coordination Workflows
1. Teacher deployment: enrollment data → optimization model → deployment recommendation
2. Resource distribution: school index → prioritized distribution list
3. Outcome tracking: exam result → longitudinal cohort analysis
